## Color tokens (example)

- **Goal**: Define semantic color roles you can map to any palette.

- **Suggested roles**:
  - `color-bg-surface`: Default surface/background.
  - `color-bg-subtle`: Subtle background for grouped content.
  - `color-fg-default`: Primary text color.
  - `color-fg-muted`: Secondary text and metadata.
  - `color-accent-primary`: Primary action and key highlights.
  - `color-border-subtle`: Neutral borders and dividers.
  - `color-border-critical`: Destructive actions and errors.

- **Example neutral palette mapping** (replace with your brand):
  - `color-bg-surface`: `#0b0b10`
  - `color-bg-subtle`: `#14141c`
  - `color-fg-default`: `#f9fafb`
  - `color-fg-muted`: `#9ca3af`
  - `color-accent-primary`: `#4f46e5`
  - `color-border-subtle`: `#1f2933`
  - `color-border-critical`: `#f97373`
