## Typography tokens (example)

- **Goal**: Define a small, consistent type scale with named roles.

- **Base settings**:
  - Font family: `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
  - Base size: `16px`
  - Line height: `1.5`

- **Roles**:
  - `heading-xl`: 28–32px, semi-bold, used for page titles.
  - `heading-m`: 20–24px, semi-bold, for section headings.
  - `body`: 16px, regular, for most content.
  - `body-small`: 14px, regular, for secondary text.
  - `caption`: 12–13px, regular, for metadata and labels.

- **Usage**:
  - Use `heading-xl` once per page as the primary title.
  - Use `heading-m` to structure sections under the main title.
  - Avoid inventing ad-hoc font sizes; map to one of these roles instead.
