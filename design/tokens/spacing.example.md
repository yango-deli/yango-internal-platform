## Spacing tokens (example)

- **Goal**: Provide a simple spacing scale to use across layout and components.

- **Scale** (you can adjust the numeric values, but keep the pattern):
  - `spacing-1`: 4px
  - `spacing-2`: 8px
  - `spacing-3`: 12px
  - `spacing-4`: 16px
  - `spacing-5`: 24px
  - `spacing-6`: 32px
  - `spacing-7`: 40px
  - `spacing-8`: 56px

- **Usage guidance**:
  - Inside components (between label and input, icon and text): use `spacing-1` to `spacing-3`.
  - Between related controls in a form: use `spacing-3` or `spacing-4`.
  - Between sections on a page: use `spacing-5` to `spacing-7`.
  - Page gutters and major layout boundaries: use `spacing-7` or `spacing-8`.

- **Notes**:
  - Avoid one-off spacing values that do not map to this scale.
  - Prefer increasing spacing between larger conceptual groups instead of adding random gaps.
